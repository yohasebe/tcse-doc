# 一時停止して確認する（Pause and Check）

トランスクリプトを常に読んでいては、リスニング力は向上しません。しかしトランスクリプトなしでは、トークの内容を追い、スピーカーのメッセージを理解するのが難しい場合もあります。特に初級・中級レベルの学習者にとってはなおさらです。TCSEの「一時停止して確認」機能は、この両方の良いところを提供します。

フルスクリーン表示モードでは、トランスクリプトと翻訳テキストは**ビデオが一時停止されたときだけ**表示されるため、画面上のテキストを読むのではなくリスニングに集中できます。不明な点や理解しにくい箇所があれば、一時停止してトランスクリプトを確認するだけです。

この機能を使うには：

1. ビデオの再生を開始する
2. [フルスクリーンモード](../playing-video/switch-to-fullscreen.md)に切り替える（`ESC`、`E`、`B` キー）
3. [スタディモード](../playing-video/pause-video-after-segment.md)を有効にする（`A` キー）と、各セグメント後に自動一時停止
