# Get unique URL of a segment

Each segment listed in search results has a unique URL that can be used to load and play the segment later. To obtain the URL, click on the **clip** icon in the search results.

![Clip icon in search results](images/01.png)

The unique URL of the segment will be presented in a popup window. You can copy and share this URL for bookmarking or sharing specific segments.

![Segment URL popup](images/02.png)
