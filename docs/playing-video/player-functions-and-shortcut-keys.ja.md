# プレーヤー機能とショートカットキー

TCSEのビデオプレーヤーは、効率的な操作のためのキーボードショートカットに対応しています。

**プレーヤーボタンとキーボードショートカット**

| 機能 | ショートカットキー | 説明 |
| :--- | :--- | :--- |
| RANDOM | — | ランダムに選ばれたトークを再生 |
| PLAY / PAUSE | `SPACE`, `K`, `P`, `PAGEDOWN` | ビデオの再生・一時停止 |
| REPEAT | `R`, `O`, `PAGEUP` | 現在のセグメントを繰り返し再生 |
| STUDY MODE | `A` | セグメント終了後に自動一時停止 |
| TRANSLATION | `T` | 翻訳の表示・非表示 |
| LISTENING MODE | `/` | トランスクリプトテキストの表示・非表示 |
| TALK INFO | `I` | トークの情報を表示 |
| FONT S / M / L | `S`, `M`, `L` | フォントサイズ変更（小・中・大） |
| FULLSCREEN | `ESC`, `E`, `B` | フルスクリーンモードと通常表示モードの切り替え |
| GO TO TOP | — | ページ上部にスクロール |
