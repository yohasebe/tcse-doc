# Player functions and shortcut keys

TCSE's video player supports keyboard shortcuts for efficient navigation and control.

**Player buttons and keyboard shortcuts**

| Function | Shortcut Keys | Description |
| :--- | :--- | :--- |
| RANDOM | — | Play a randomly selected talk |
| PLAY / PAUSE | `SPACE`, `K`, `P`, `PAGEDOWN` | Play or pause the video |
| REPEAT | `R`, `O`, `PAGEUP` | Replay the current segment |
| STUDY MODE | `A` | Auto-pause after each segment |
| TRANSLATION | `T` | Show or hide translation |
| LISTENING MODE | `/` | Show or hide transcript text |
| TALK INFO | `I` | Show information about the talk |
| FONT S / M / L | `S`, `M`, `L` | Change font size (small, medium, large) |
| FULLSCREEN | `ESC`, `E`, `B` | Toggle fullscreen and normal viewing modes |
| GO TO TOP | — | Scroll to top of page |
