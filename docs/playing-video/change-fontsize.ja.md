# トランスクリプトのフォントサイズを変更する

ビデオコントローラーの `S`、`M`、`L` ボタンをクリックするか、対応するキーボードショートカット（それぞれ `S`、`M`、`L`）を使って、トランスクリプトテキストのフォントサイズを変更できます。
