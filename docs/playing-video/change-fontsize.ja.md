# トランスクリプトのフォントサイズを変更する

ビデオコントローラーの `S`、`M`、`L` ボタンをクリックするか、対応するキーボードショートカット（それぞれ `S`、`M`、`L`）を使って、トランスクリプトテキストのフォントサイズを変更できます。フォントサイズは通常表示モードとフルスクリーン表示モードの両方に適用されます。
