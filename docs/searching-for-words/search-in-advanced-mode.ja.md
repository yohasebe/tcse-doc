# アドバンスト・サーチで検索する

1. [アドバンスト・サーチのクエリ構文](advanced-search-query-syntax.md)で検索文字列を入力する
2. **Search Target** が **English Transcript** に設定されていることを確認する
3. **Advanced Search** にチェックを入れる
4. **SEARCH** をクリックする

![検索結果の例](images/08.png)

アドバンスト・サーチでは、品詞タグ、レンマ、依存関係、形態素情報、固有表現、ワイルドカードなどが使用できます。詳しくは[クエリ構文リファレンス](advanced-search-query-syntax.md)を参照してください。
