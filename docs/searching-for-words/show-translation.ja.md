# 対訳を表示する

1. 検索文字列を入力する
2. **Translation** ドロップダウンから対訳言語を選択する
3. **Search Target** が **English Transcript** に設定されていることを確認する
4. 翻訳のあるトークだけに限定する場合は **Include English only talks** のチェックを外す
5. **SEARCH** をクリックする

![入力パネル](images/01.png)

![検索結果の例](images/02.png)

選択した言語の翻訳テキストが、英語の検索結果の横に表示されます。

TCSEでは以下の34言語の翻訳に対応しています：

アラビア語、イタリア語、インドネシア語、ウクライナ語、オランダ語、ギリシャ語、クルド語（中央）、クルド語（北部）、クロアチア語、スウェーデン語、スペイン語、スロバキア語、セルビア語、タイ語、チェコ語、中国語（簡体字）、中国語（繁体字）、ドイツ語、トルコ語、日本語、ハンガリー語、ヒンディー語、ビルマ語、フランス語、ブルガリア語、ヘブライ語、ベトナム語、ペルシア語、ポーランド語、ポルトガル語、ポルトガル語（ブラジル）、ルーマニア語、ロシア語、韓国語

すべてのトークにすべての言語の翻訳があるわけではありません。利用可能なトーク数は言語によって異なります。
