# 対訳を表示する

1. 検索文字列を入力する
2. **Translation** ドロップダウンから対訳言語を選択する
3. **Search Target** が **English Transcript** に設定されていることを確認する
4. 翻訳のあるトークだけに限定する場合は **Include English only talks** のチェックを外す
5. **SEARCH** をクリックする

![入力パネル](images/01.png)

![検索結果の例](images/02.png)

選択した言語の翻訳テキストが、英語の検索結果の横に表示されます。

TCSEでは **34言語** の翻訳に対応しています。一覧は[よくある質問](../faq.md#what-are-the-criteria-of-translation-language-choice)を参照してください。すべてのトークにすべての言語の翻訳があるわけではありません。利用可能なトーク数は言語によって異なります。
