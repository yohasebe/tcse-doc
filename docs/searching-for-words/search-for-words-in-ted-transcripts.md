# Search in transcripts

1. Input a search string
2. Click on **SEARCH**

![Input Panel](images/01.png)

![Sample Results](images/02.png)

By default, TCSE uses **expanded segments** — units that contain at least one complete sentence. This provides more natural context for each search result.

Results are paginated with page number buttons at the top and bottom of the results area. Use the **Prev** and **Next** buttons or click on a specific page number to navigate.

You can also switch to [KWIC concordance view](kwic-concordance-view.md) for a compact, linguistically-oriented display of results.
