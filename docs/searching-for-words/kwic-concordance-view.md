# KWIC concordance view

KWIC (Key Word In Context) is a concordance display format commonly used in corpus linguistics. It shows each search result with the keyword centered, surrounded by its left and right context.

## Switching between List and KWIC views

After performing a search, click the **KWIC** button to switch from the default list view to the KWIC concordance view. Click **List** to switch back.

![KWIC toggle button](images/09.png)

## KWIC display format

The KWIC view presents results in a three-column table:

| Column | Content |
| :--- | :--- |
| **Left context** | Words preceding the keyword (right-aligned) |
| **Keyword** | The matched search term (centered, highlighted) |
| **Right context** | Words following the keyword (left-aligned) |

Each row also includes icons for viewing the segment in context and playing the video.

![KWIC concordance view](images/10.png)

## Compact display

For advanced searches or queries with three or more words, keywords are shown in a compact format with a placeholder marker (`···`). Hover over the marker to see the full context, or click it for details.

## Tips

- The KWIC view is available for all search types (regular and advanced)
- Pagination works the same way as in list view
- Switching between List and KWIC does not trigger a new search
