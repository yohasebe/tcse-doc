# 固有表現検索

TCSEでは、spaCyのNER（固有表現認識）システムが認識した固有表現を検索できます。アドバンスト・サーチで `%ENTITY` 記法を使用して、TED Talkトランスクリプト内の特定の種類の固有表現を検索します。

## 使い方

1. `%ENTITY` 記法でクエリを入力する（例: `%PERSON said`）
2. **Advanced Search** にチェックを入れる
3. **SEARCH** をクリックする

## エンティティタイプ

TCSEは18種類の固有表現タイプを認識します：

| エンティティタイプ | 説明 | コーパス内の件数 |
| :--- | :--- | ---: |
| `%DATE` | 日付（絶対・相対）や期間 | 167,347 |
| `%CARDINAL` | 他のタイプに該当しない数値 | 96,042 |
| `%PERSON` | 人名（架空の人物を含む） | 87,068 |
| `%ORG` | 企業、機関、組織 | 80,776 |
| `%GPE` | 国、都市、州 | 61,648 |
| `%NORP` | 国籍、宗教・政治団体 | 23,859 |
| `%TIME` | 1日未満の時刻 | 22,649 |
| `%ORDINAL` | "first"、"second" 等の序数 | 21,850 |
| `%LOC` | GPE以外の地名（山脈、水域など） | 21,747 |
| `%PERCENT` | パーセンテージ（「%」を含む） | 20,511 |
| `%QUANTITY` | 計量単位（重量、距離など） | 19,442 |
| `%WORK_OF_ART` | 書籍、楽曲等のタイトル | 16,235 |
| `%MONEY` | 金額 | 15,118 |
| `%FAC` | 建物、空港、高速道路、橋 | 6,719 |
| `%EVENT` | ハリケーン、戦争、スポーツイベント等の名称 | 5,928 |
| `%PRODUCT` | 製品、乗り物、食品（サービス以外） | 4,286 |
| `%LAW` | 法律として成立した文書の名称 | 2,304 |
| `%LANGUAGE` | 言語名 | 1,560 |

## 使用例

| クエリ | 検索対象 |
| :--- | :--- |
| `%PERSON` | すべての人名 |
| `%PERSON said` | 人名の後に "said" が続くもの |
| `in %GPE` | "in" の後に国名・都市名が続くもの |
| `%MONEY` | すべての金額表現 |
| `%DATE` | すべての日付表現 |
| `%ORG [be]` | 組織名の後に "be" の活用形が続くもの |
