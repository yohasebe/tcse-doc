# Video type filter

TCSE allows you to filter search results by video type. TED Talks are categorized into different types (e.g., TED, TEDx, TED-Ed, etc.), and you can select which types to include in your search.

## How to use

In the configuration panel, you will see **Video Types** checkboxes with the number of talks available for each type. Check or uncheck the types you want to include or exclude from your search results.

- **Check All**: Select all video types
- **Uncheck All**: Deselect all video types

The number displayed next to each video type indicates how many talks of that type are in the database.

![Video type filter](images/13.png)

!!! tip "Tips"
    - By default, all video types are selected
    - Filtering by video type affects all search modes (regular, advanced, talk info)
    - The total number of talks matching your filter is shown in the results
