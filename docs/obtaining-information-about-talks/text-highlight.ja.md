# テキストハイライト

全文トランスクリプト表示では、重要な語句をハイライトして、主要な内容や談話構造を一目で把握できます。

## ハイライトモード

ハイライトは**デフォルトでオフ**になっており、トランスクリプトがすっきりした状態で表示されます。全文トランスクリプト表示の上部にあるチェックボックスをオンにして、ハイライトを有効にしてください。

### キーワード

現在のトークにおけるTF-IDFスコアが高い（3.0超）内容語をハイライトします。これらはコーパス全体と比較して、そのトークで統計的に有意な語であり、トークの主要なトピックや用語を素早く把握するのに役立ちます。

### ディスコースマーカー

ディスコースマーカー（トークの構造や流れを示す語句。例: 転換、緩衝表現、強調マーカーなど）をハイライトします。スピーカーがプレゼンテーションをどのように構成しているかを学ぶのに役立ちます。

![テキストハイライトコントロールとハイライトされたトランスクリプト](images/14.png)

!!! tip "ヒント"
    - ハイライトはデフォルトでオフ — チェックボックスをオンにして有効化
    - キーワードとディスコースマーカーは同時にハイライト可能
    - ハイライトは通常セグメントモードと拡張セグメントモードの両方で利用可能
    - ハイライトの色でキーワードとディスコースマーカーを区別できる
