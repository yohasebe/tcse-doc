# Find basic information about a talk

Click either on the ID number or the speaker/title of the talk to open the talk information modal.

![Where to click](images/01.png)

![Talk information modal](images/02.png)

The talk information includes:

- Speaker name and biographical information
- Talk title and description
- Duration and number of segments
- Available translations
- Video type and date
- Links to the original TED page
