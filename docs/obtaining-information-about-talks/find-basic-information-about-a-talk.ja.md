# トークの基本情報を表示する

検索結果の **トークID番号** をクリックすると、トーク情報のモーダルが開きます。（トーク検索の結果では、スピーカー名やタイトルのクリックでも開きます。）

![クリック位置](images/01.png)

![トーク情報モーダル](images/02.png)

トーク情報モーダルには以下の情報が含まれます：

- トークのタイトル（英語、および翻訳がある場合はその翻訳）
- **Watch this talk on TCSE** リンク
- サムネイル画像
- **Recorded on**（収録日）
- **Event**（イベント名、例: TEDWomen 2017）
- **Number of elements**（トークン数）
- **Number of segments**（セグメント数）
- **Number of sentences**（文数）
- **Duration**（所要時間、分:秒）
- **Words per minute**（1分あたりの語数）と速度指標（very slow / slow / moderate / fast / very fast）
- **Readability (FK)**（Flesch-Kincaid読みやすさスコア）と難易度指標
- **Vocabulary (CEFR)**（語彙レベル分布バー、A1〜C2）
- **Syntax Profile**（統語プロファイル）: NP複雑度、受動態の割合、関係節の割合、従属節の割合
- **Video Type**（ビデオタイプ、例: TED Stage Talk, TED-Ed Original）
- **Official TED page** リンク
- **Speaker information from Wikipedia**（利用可能な場合）: 名前、生年月日、性別、国籍、職業、説明
- **Synopsis**（トークの説明文、英語および翻訳がある場合はその翻訳）

## 語彙プロファイル（CEFR）

**Vocabulary (CEFR)** の行には、CEFR（ヨーロッパ言語共通参照枠）の6段階に基づく語彙分布を示す色分けバーが表示されます：

| レベル | 色 | 説明 |
| :--- | :--- | :--- |
| **A1** | 緑 | 初級 — 最も基礎的な語彙 |
| **A2** | 薄緑 | 初級上 |
| **B1** | 黄 | 中級 |
| **B2** | オレンジ | 中級上 |
| **C1** | 赤 | 上級 |
| **C2** | 紫 | 最上級 — 最も高度な語彙 |

各色のセグメントの幅は、そのレベルのトークンの割合に比例しています。セグメントにマウスカーソルを合わせると、正確な割合・トークン数・タイプ数が表示されます（例: 「B1: 25.3% (520 tokens / 180 types)」）。

緑・薄緑の領域が広いトークはより平易な語彙を使用しており、オレンジ・赤・紫の領域が多いトークはより高度な語彙を使用しています。学習者のレベルに合ったトーク選びや、教育者によるテキスト難易度の評価に役立ちます。

## 統語プロファイル

**Syntax Profile** の行には、トークの統語的複雑さを特徴づける4つの指標が表示されます：

| 指標 | 説明 |
| :--- | :--- |
| **NP complexity** | 名詞句チャンクあたりの平均語数。値が大きいほど、修飾語の多い複雑な名詞句が多いことを示す。 |
| **Passive voice** | 受動態の使用率（%）。学術的・フォーマルなトークほど高くなる傾向がある。 |
| **Relative clauses** | 関係節の割合（%）。「the person *who spoke*」のような構造。値が高いほど文の埋め込みが複雑。 |
| **Subordination** | 従属節の割合（%）。「*because* ...」「*although* ...」などの構造。値が高いほど文構造が複雑。 |

これらの指標は、読みやすさスコアやCEFR語彙プロファイルと組み合わせることで、トークの言語的複雑さの全体像を把握できます。
